# BAMS (Blockchain Attendance Management System)

A decentralized attendance management system built on blockchain technology, designed for educational institutions to securely record and verify student attendance.

## Features

- **Secure & Transparent**: Leverages blockchain for tamper-proof attendance records
- **Real-time Tracking**: Live attendance monitoring and reporting
- **User Roles**: Multiple user roles (Admin, Teacher, Student)
- **Visual Hierarchy**: Interactive 3D visualization of organizational structure
- **Comprehensive Reporting**: Detailed attendance reports and analytics

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript, Three.js
- **Backend**: Node.js, Express.js
- **Blockchain**: Ethereum (or other compatible blockchain)
- **Database**: JSON-based storage (can be extended to IPFS/blockchain)
- **Authentication**: JWT (JSON Web Tokens)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `npm start`
4. Open `http://localhost:3000` in your browser
